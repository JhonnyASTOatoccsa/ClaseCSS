## historia
en un principio era parte de htmll como atributo de tipo de estilo y su atributo se llamaba "style".
"style"se usaba para darle estilos como formato y color a la etiqueta que hacia uso de este atributo.
"style" este atributo se usaba a travez de una lista de caracteristicas cada caracteristica se ingresaba por una clave y un valor
***ejmplo***
```css
<h1 style="color:red;font-size:45px">hola amigos<h/1>
```
